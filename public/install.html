Install

Bower

bower install skitter --save

Manually

Download zip and move files in dist directory for you application. Attention: you need to download the dependencies manually.

Head

<link href="dist/skitter.css" type="text/css" media="all" rel="stylesheet" />

<script src="/js/jquery-2.1.1.min.js"></script>
<script src="/js/jquery.easing.1.3.js"></script>
<script src="/dist/jquery.skitter.min.js"></script>
